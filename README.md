# 📈 디지털 마케팅 전환율 예측 및 AI 전략 제언 시스템
> **DNN 기반 잠재 고객 분류와 LLM 마케팅 제언을 결합한 하이브리드 지능형 솔루션**
> 
> 고객의 행동 데이터를 분석하여 구매 전환 확률을 예측하고, AI가 해당 고객을 위한 맞춤형 마케팅 전략을 자동으로 생성해 주는 서비스입니다.

---

## 📌 프로젝트 개요
* **진행 기간:** 2025.12.30 ~ 2026.01.07
* **핵심 목표:** * 고도화된 딥러닝 모델을 통한 고가치 타겟 고객 선별
    * 마케팅 리소스 최적화를 위한 데이터 기반 의사결정 지원
    * 예측 결과에 대한 LLM의 비즈니스 관점 해석 제공

## 🛠 기술 스택
- **Language:** ![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
- **Deep Learning:** TensorFlow, Keras (DNN), Scikit-learn
- **LLM:** Google Gemini API (마케팅 전략 제언)
- **Framework:** Streamlit (프로토타입 구현)
- **Data:** Pandas, NumPy, Matplotlib, Seaborn

## ✨ 주요 특징 (Hybrid Intelligence)
### 1. 딥러닝 기반 전환 확률 예측 (DNN)
- 16개의 고객 행동 지표(CTR, 체류 시간, 페이지 뷰 등)를 입력받아 다차원 비선형 관계 분석
- 이진 분류(Binary Classification)를 통해 0~1 사이의 정밀한 구매 전환 확률 산출
- 높은 예측 성능을 바탕으로 실제 구매 가능성이 높은 잠재 고객 선별

### 2. LLM 비즈니스 통역사 (Gemini 연동)
- 정형 데이터(숫자)인 예측 결과를 비정형 데이터(언어)로 변환
- 분석된 고객 특성에 맞춰 '개인화된 마케팅 문구' 및 '채널별 전략' 자동 서술

### 3. 시스템 안정성 (Double-Path Strategy)
- 외부 API 장애 발생 시에도 시스템이 멈추지 않도록 내부 로직(Rule-based)으로 즉시 전환되는 Fallback 시스템 구축

## 🏗 시스템 아키텍처


1. **Input**: 실시간 고객 행동 데이터 입력
2. **Analysis**: DNN 모델을 통한 전환 가능성 점수화
3. **Reasoning**: 점수 및 변수 중요도(Feature Importance)를 바탕으로 LLM이 판단 근거 분석
4. **Action**: 마케터에게 최적화된 대응 전략 제언

## 🚀 주요 기능 및 UI
- **예측 대시보드:** 사이드바에 고객 정보 입력 시 실시간으로 구매 전환 가능성 계산
- **시각화 도구:** - 임계값 기준 차트를 통한 타겟팅 가이드라인 제시
    - 모델의 변수 중요도 그래프 시각화 (어떤 행동이 전환에 큰 영향을 주었는지 파악)
- **AI 맞춤형 제언:** 입력값과 결과에 따른 AI의 전략적 피드백 제공
